# Guna Teddi | Developer Portfolio
 
A modern, responsive developer portfolio website built to showcase projects, skills, and professional experience. Designed with a focus on clean aesthetics, accessibility, and seamless user experience.

## ✨ Features

- **Responsive Design** - Optimized for all screen sizes from mobile to desktop
- **Dark/Light Theme** - Automatic theme detection with manual toggle
- **Smooth Animations** - Scroll-triggered reveals and transitions
- **Accessibility** - Skip links, semantic HTML, and ARIA labels
- **Performance Optimized** - Fast loading with minimal dependencies
- **Mobile-Friendly Navigation** - Collapsible hamburger menu
- **Project Showcase** - Grid layout displaying featured projects
- **Contact Form** - Ready for backend integration (Formspree/Netlify Forms)

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, grid, animations
- **JavaScript** - Theme switching, navigation, scroll effects
- **FontShare** - Satoshi & Cabinet Grotesk fonts
- **SVG** - Custom icons and illustrations

## 📁 Project Structure

portfolio/

├── style.css # All styles (included in HTML for single-file version)
├── script.js # JavaScript functionality (included in HTML)
└── README.md # Documentation

### Prerequisites
- Any modern web browser
- Code editor (VS Code recommended)
- Basic knowledge of HTML/CSS/JS for customization
